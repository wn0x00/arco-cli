# @guanzhu.me/arco-cli

Initialize projects from selectable Arco templates.

## Install

```bash
npm install @guanzhu.me/arco-cli -g
```

## Usage

```bash
arco init my-app
```

Use a specific template package:

```bash
arco init my-app --template arco-design-pro
```

Use a local template:

```bash
arco init my-app --template file:../my-template
```
